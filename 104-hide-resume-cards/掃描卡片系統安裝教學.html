新人安裝指南

掃描卡片系統安裝教學

請依序完成 Chrome 擴充功能安裝、開啟使用者指令碼權限、貼上公司腳本、重新整理頁面。整個流程約 5 分鐘。

安裝 Tampermonkey 開啟公司腳本

Chrome 內部管理頁需貼到網址列開啟，無法從一般網頁直接跳轉。

完成後確認

  • 1Tampermonkey 已安裝並固定在 Chrome 工具列
  • 2「允許使用者指令碼」已打開
  • 3公司腳本已完整貼上並儲存
  • 4腳本狀態為啟用，掃描卡片頁面已重新整理

建議使用 Google Chrome 操作。若畫面與教學不同，請先截圖回報系統窗口。

1

安裝篡改猴

  1. 點上方「安裝 Tampermonkey」。
  2. 確認頁面名稱為 Tampermonkey。
  3. 點選「加到 Chrome」或「Add to Desktop」。
  4. 跳出確認視窗後，點選「新增擴充功能」。
  5. 點 Chrome 右上角拼圖圖示，將 Tampermonkey 固定在工具列。
2

開啟腳本權限

  1. 點上方「複製擴充功能管理網址」，貼到 Chrome 網址列後按 Enter；也可以手動輸入 chrome://extensions/。
  2. 找到 Tampermonkey 或「篡改猴」卡片。
  3. 點卡片左下角「詳細資料」。
  4. 往下找到「允許使用者指令碼」。
  5. 將右側開關打開，確認開關變成藍色或啟用狀態。
3

貼上公司腳本

  1. 點 Tampermonkey 圖示。
  2. 選擇「管理面板」或「Dashboard」。
  3. 點選「＋」新增腳本。
  4. 刪除畫面中原本的範例程式碼。
  5. 開啟公司腳本，複製完整內容後貼上。
  6. Windows 按 Ctrl + S，Mac 按 Command + S 儲存。
4

確認功能

  1. 回到 Tampermonkey 管理面板。
  2. 確認腳本狀態為「已啟用」。
  3. 開啟或重新整理掃描卡片系統頁面。
  4. 確認頁面上出現卡片篩選或隱藏相關功能。

貼上時請注意

請複製完整腳本內容，不要只複製 GitHub 網址。若 GitHub 頁面有「Raw」按鈕，可以先點 Raw，再全選並複製內容。第一次使用前也請確認 Tampermonkey 詳細資料中的「允許使用者指令碼」已打開。

常見狀況排除

  • 找不到 Tampermonkey：請點 Chrome 右上角拼圖圖示，確認是否已安裝。
  • 無法執行腳本：請到 chrome://extensions/ 找到 Tampermonkey，進入「詳細資料」並打開「允許使用者指令碼」。
  • 找不到新增腳本：請進入 Tampermonkey 的管理面板，再點選「＋」。
  • 功能沒有出現：請確認腳本已儲存、狀態已啟用，並重新整理掃描卡片系統頁面。
  • 仍無法使用：請截圖目前畫面，回報 HR 或系統窗口協助確認。
